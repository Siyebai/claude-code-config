## 推荐算法

`推荐量 = f(内容质量 × 用户兴趣匹配 × 互动信号 × 时效性)`
**内容质量+用户真实互动 占45%权重**（2025清风整改后）

三阶段: 冷启动(500-5000人) → 数据评估(点击/完读/互动/时长) → 流量扩展/停止

关键规则:
- 95%推潜在兴趣用户, 5%推粉丝
- 3000字密集信息推荐周期72小时
- 搜索进入→互动→后续同类优先推荐
- 粉丝阅读占比>60%降权（防圈层固化）
- AI段落指纹重复>40%降权

## 微头条爆款公式

| 公式 | 结构 | 关键 |
|------|------|------|
| 反常识钩子 | 常见认知+反常识+悬念 | 前30字决定60% |
| 情绪叙事 | 场景+共鸣+提问 | 具体场景引发互动 |
| 信息差 | 小众身份+行业内幕 | 盐选审稿潜规则类 |
| 场景痛点 | 场景+痛点+经验+讨论 | 换角度写作类 |

**评论钩子**: 发布30分钟内主号发争议/提问式回复 → 评论率0.8%→5%+

**最佳时段**: 06-08通勤 / 11-13午休黄金 / 17-19下班 / 21-23睡前 / 01-03凌晨冷启动

## 文章标题公式

- 数字+结果: 「写了47天赚了382块，但昨天一篇198」
- 反常识+疑问: 「你以为是文笔不过稿？真相可能跟写作无关」
- 身份+经历: 「一个写了三年悬疑的新人，走过的全部弯路」
- 悬念+信息差: 「盐选不告诉你的三个审稿潜规则」
- 对比+转折: 「从被拒12次到一篇过稿6万展现」

**5+3+2关键词布局**: 标题5核心词→前300字3场景长尾词→结尾2关联词（识别率+70%）

**选题坐标系**: X=垂直深度 Y=痛点层级 Z=时间衰减 → 长尾选题冷启动是泛选题3-5倍

## 冷启动节奏 (3+7+21)

- 前3天: 每天1微头条，测试标签
- 第4-10天: 每天1图文+1微头条，建立内容库
- 第11-31天: 保持日更，每条解决一个具体问题

**内容配比**: 小说片段30% + 写作干货30% + 日常感悟20% + 热点20%

**三大红线**:
1. 不碰营销词（链接/优惠/下单）
2. AI段落指纹>40%降权→加个人案例+反向论证
3. 同WiFi≥3账号被标记营销矩阵
